# Training Bridge 🎓

A comprehensive mobile training management system connecting university students, training institutions, and university administration.

## Features ✨

### For Students
- 📱 Browse training opportunities
- 📝 Apply with ease
- 📊 Track application status
- 📄 Submit weekly reports
- 💼 Manage internship progress

### For Institutions
- 🏢 Post training opportunities
- 👥 Manage applicants
- ✅ Accept/Reject applications
- 📈 Track intern performance
- ⭐ Evaluate interns

## Tech Stack 🛠️

- **Framework**: Flutter
- **State Management**: Provider
- **API Communication**: HTTP
- **Storage**: Shared Preferences, Secure Storage
- **UI/UX**: Material Design with Glassmorphism
- **Notifications**: Firebase Cloud Messaging
